Comandos alem do PopupMenu:

/server
/join /j
/motd
/part
/msg
/nick
/quit
/disconnect
/seu comando

/destino
/senha

Alt+I     ---- Abre janela de informações
Alt+L     ---- Abre janela com Lista de nicks
Alt+D     ---- Permuta entre modo DOS e modo IRC
Enter     ---- Envia texto
Setas     ---- Cima/Baixo mostra mensagens anteriores

Destino é para onde as mensagens serão enviadas(Nick/#Canal)

Modo DOS aceita apenas comandos que usam aplicativos
Ex.:
 arp -a
 netstat
